    Homotopies

    Let's consider the type theory represent of proposition "$f: A ‚Üí B$ is surjective":

    $$ \Pi_{(y:B)}\Sigma_{(x:A)} f(x) = y $$

    Note $=$ (or we should use $=_B$) here stands for this is an instance of "identity type".

    Apply AC we introduced in last course:

    $$ \Sigma_{(g: B\rightarrow A)} \Pi_{(y:B)} f(g(y)) = y $$

    So we can see there's some relation between $f \cdot g$ and $id_B$, this relation is similar, but weaker than $f \cdot g \underset{B ‚Üí B}{=} id_B$.

    And this relation is what we call homotopy.

    Formal definition

    For two dependent functions $f$ and $g$,

    $$ f \sim g := \Pi_{(x:A)} f(x) = g(x) $$

    Here $f \sim g$ is the type of homotopies from $f$ to $g$.

    module _ {l1 l2} {A : Set l1} {B : A ‚Üí Set l2} where
    _~_ : (f g : (x : A) ‚Üí B x) ‚Üí Set (l1 ‚äî l2)
    f ~ g = (x : A) ‚Üí f x ‚â° g x

    Here we can say $g$ is a section of $f$ and $f$ is a retraction of $g$, we'll introduce these concepts later.

    Example

    neg-bool : Bool ‚Üí Bool
neg-bool true = false
neg-bool false = true

    There's no way can we prove:

    $$ neg\text{-}bool \cdot neg\text{-}bool ‚âê id_{Bool} $$

    Instead, we can define a pointwise identification between the values of $neg\text{-}bool \cdot neg\text{-}bool$ and $id_{Bool}$.

    neg-neg-bool : (neg-bool ‚àò neg-bool) ~ id
neg-neg-bool false = refl
neg-neg-bool true = refl

    Commutativity of Diagrams1

    We can use homotopies to express the commutativity of diagrams.

    For example, we say that a triangle commutes if it comes equipped with a homotopy $H: f \sim g \cdot h$.

    $$ \require{amscd} \require{cancel} \def\diaguparrow#1{\smash{\raise.6em\rlap{\ \ \scriptstyle #1} \lower.6em{\cancelto{}{\Space{2em}{1.7em}{0px}}}}} \begin{CD} A @>f>>X\\ @VhVV \diaguparrow{g}\\ B \end{CD} $$

    A square is similar:

    $$ \require{amscd} \begin{CD} A @>g>> A'\\ @VfVV @Vf'VV\\ B @>h>> B' \end{CD} $$

    Here $h \cdot f \sim f' \cdot g$.

    And we can have homotopies between homotopies:

    If $H, K: f \sim g$ are two homotopies, ie

    $$ f,g: \Pi_{(x:A)}B(x) $$ $$ H,K: f \sim g $$

    then the type of homotopies $f \sim g$ between them is just the type

    $$ \alpha: \Pi_{(x:A)}H(x) = K(x) $$

    And we can push this further, homotopies of homotopies of homotopies, homotopies4, etc.

    Operations on Homotopies

    refl-htpy: $\Pi_{(f:\Pi_{(x:A)}B(x))} f \sim f$

    inv-htpy: $\Pi_{(f,g:\Pi_{(x:A)}B(x))} (f \sim g) ‚Üí (g \sim f)$

    concat-htpy: $\Pi_{(f,g,h:\Pi_{(x:A)}B(x))} (f \sim g) ‚Üí (g \sim h) \rightarrow (f \sim h)$

    We will often write $H^{-1}$ for $inv\text{-}htpy(H)$, and $H \cdot K$ for $concat\text{-}htpy(H, K)$

    refl-htpy : {A : Set} {B : A ‚Üí Set} (f : (x : A) ‚Üí B x) ‚Üí f ~ f
refl-htpy = Œª f x ‚Üí refl

inv-htpy : {A : Set} {B : A ‚Üí Set} {f g : (x : A) ‚Üí B x} ‚Üí f ~ g ‚Üí g ~ f
inv-htpy htpy = Œª x ‚Üí sym (htpy x)

concat-htpy : {A : Set} {B : A ‚Üí Set} {f g h : (x : A) ‚Üí B x} ‚Üí f ~ g ‚Üí g ~ h ‚Üí f ~ h
concat-htpy htpy1 htpy2 = Œª x ‚Üí trans (htpy1 x) (htpy2 x)

_¬∑_ : {A : Set} {B : A ‚Üí Set} {f g h : (x : A) ‚Üí B x} ‚Üí f ~ g ‚Üí g ~ h ‚Üí f ~ h
_¬∑_ = concat-htpy

    Laws of Homotopies

    Homotopies satisfy the groupoid laws, ie. associative, has left and right unit (refl-htpy), left and right inverse law.

    assoc-htpy : {A B : Set} {f g h i : A ‚Üí B} (H : f ~ g) (K : g ~ h) (L : h ~ i) ‚Üí (((H ¬∑ K) ¬∑ L) ~ (H ¬∑ (K ¬∑ L)))
assoc-htpy H K L = Œª x ‚Üí trans-assoc (H x) {K x} {L x}

left-unit-htpy : {A B : Set} {f g : A ‚Üí B} (H : f ~ g) ‚Üí ((refl-htpy f ¬∑ H) ~ H)
left-unit-htpy H = Œª x ‚Üí refl

right-unit : {A : Set} {x y : A} {p : x ‚â° y} ‚Üí (trans p (refl {x = y})) ‚â° p
right-unit {p = refl} = refl

right-unit-htpy : {A B : Set} {f g : A ‚Üí B} (H : f ~ g) ‚Üí ((H ¬∑ (refl-htpy g)) ~ H)
right-unit-htpy {_} {_} {f} {g} H = Œª x ‚Üí right-unit

right-inv : {A : Set} {x y : A} (p : x ‚â° y) ‚Üí (trans p (sym p)) ‚â° refl
right-inv refl = refl

left-inv : {A : Set} {x y : A} (p : x ‚â° y) ‚Üí (trans (sym p) p) ‚â° refl
left-inv refl = refl

left-inv-htpy : {A B : Set} {f g : A ‚Üí B} (H : f ~ g) ‚Üí (((inv-htpy H) ¬∑ H) ~ (refl-htpy g))
left-inv-htpy H = Œª x ‚Üí left-inv (H x)

right-inv-htpy : {A B : Set} {f g : A ‚Üí B} (H : f ~ g) ‚Üí ((H ¬∑ (inv-htpy H)) ~ (refl-htpy f))
right-inv-htpy H = Œª x ‚Üí right-inv (H x)

    Whiskering

    Whiskering operations are operations that allow us to compose homotopies with functions.

    • Suppose $H: f \sim g$ for two functions $f,g: A ‚Üí B$, and let $h:B‚ÜíC$. We define

    $$ h \cdot H := Œªx.ap_h(H(x)) : h \cdot f \sim h \cdot g $$

    f-chain-H : {A B C : Set} {f g : A ‚Üí B} (h : B ‚Üí C) (H : f ~ g) ‚Üí ((h ‚àò f) ~ (h ‚àò g))
f-chain-H h H = Œª x ‚Üí cong h (H x)
    • Suppose $f: A ‚Üí B$ and $H: g \sim h$ for two functions $g,h:B‚ÜíC$. We define

    $$ H \cdot f := Œªx.H(f(x)) : h \cdot f \sim g \cdot f $$

    H-chain-f : {A B C : Set} {g h : B ‚Üí C} (H : g ~ h) (f : A ‚Üí B) ‚Üí ((h ‚àò f) ~ (g ‚àò f))
H-chain-f H f = Œª x ‚Üí sym (H (f x))

    Bi-invertible maps

    We can use homotopies to define sections, retractions and equivalence of a map.

    • A section is a right inverse of some morphism.

      $$ sec(f) := \Sigma_{(g:B\rightarrow A)} f\cdot g \sim id_B $$

      For any equivalence $e : ùê¥ ‚âÉ ùêµ$ we define $e^{-1}$ to be the section of $e$.

      sec : {l1 l2 : Level} {A : Set l1} {B : Set l2} (f : A ‚Üí B) ‚Üí Set (l1 ‚äî l2)
sec {_} {_} {A} {B} f = Œ£ (B ‚Üí A) (Œª g ‚Üí ((f ‚àò g) ~ id))

    • A retraction is a left inverse of some morphism.

      $$ retr(f) := \Sigma_{(h:B\rightarrow A)} h\cdot f \sim id_A $$

      retr : {l1 l2 : Level} {A : Set l1} {B : Set l2} (f : A ‚Üí B) ‚Üí Set (l1 ‚äî l2)
retr {_} {_} {A} {B} f = Œ£ (B ‚Üí A) (Œª h ‚Üí ((h ‚àò f) ~ id))

    • equivalence = section + retraction

      $$ is\text{-}equiv(f) := sec(f) √ó retr(f) $$

      is-equiv : {l1 l2 : Level} {A : Set l1} {B : Set l2} (f : A ‚Üí B) ‚Üí Set (l1 ‚äî l2)
is-equiv {_} {_} {A} {B} f = sec f √ó retr f

      Or "f is bi-invertible".

      We will write $A ‚âÉ B$ for the type $\Sigma_{(f:A‚ÜíB)} is\text{-}equiv(f)$ of all equivalences from $A$ to $B$.

    • Besides, we can define $has\text{-}inverse$:

      $$ has\text{-}inverse(f) := \Sigma_{(g:B‚ÜíA)} (f\cdot g \sim id_B) \times (g\cdot f \sim id_A) $$

      has-inverse : {l1 l2 : Level} {A : Set l1} {B : Set l2} (f : A ‚Üí B) ‚Üí Set (l1 ‚äî l2)
has-inverse{_} {_} {A} {B} f = Œ£ (B ‚Üí A) (Œª g ‚Üí ((f ‚àò g) ~ id) √ó ((g ‚àò f) ~ id))

    We can prove that $has\text{-}inverse(f) \leftrightarrow is\text{-}equiv(f)$

    is-equiv-to-has-inverse : {A : Set} {B : Set} {f : A ‚Üí B} ‚Üí is-equiv f ‚Üí has-inverse f
is-equiv-to-has-inverse {f = f} ((r-inv , r-homo), (l-inv , l-homo)) = 
    r-inv , (r-homo , 
        inv-htpy (inv-htpy 
            l-homo ¬∑ (
                H-chain-f (H-chain-f l-homo r-inv ¬∑ f-chain-H l-inv r-homo) f
            )
        )
    )

has-inverse-to-is-equiv : {A : Set} {B : Set} {f : A ‚Üí B} ‚Üí has-inverse f ‚Üí is-equiv f
has-inverse-to-is-equiv (inv , (r-homo , l-homo)) = (inv , r-homo) , (inv , l-homo)

    Laws & operations on Equivalences

    We have $refl\text{-}eqiv$, $inv\text{-}eqiv$ and $concat\text{-}eqiv$.

    Function Extensionality & Univalence Axiom

    Let $P := \Pi_{(x:A)}B(x)$, because both $\sim$ and $‚âÉ$ are refl. we get

    $$ id\text{-}to\text{-}\sim: \Pi_{(f,g:P)}(f \underset{P}{=} g \rightarrow f \sim g) $$

    $$ id\text{-}to\text{-}‚âÉ: \Pi_{(A,B:\mathcal{U})}(A\underset{\mathcal{U}}{=}B \rightarrow A ‚âÉ B) $$

    With these, we can define function extensionality:

    The principle of functional extensionality states that two functions are equal if their values are equal at every argument. ‚Äî‚Äî nLab

    $$ funext:\Pi_{(f,g:P)}is\text{-}equiv(id\text{-}to\text{-}\sim(f, g)) $$

    Similarly, we can define the univalence axiom (UA):

    In intensional type theory, identity types behave like path space objects; this viewpoint is called homotopy type theory. This induces furthermore a notion of homotopy fibers, hence of homotopy equivalences between types.

    On the other hand, if type theory contains a universe Type, so that types can be considered as points of Type, then between two types we also have an identity type Paths $Paths_{Type}(X,Y)$. The univalence axiom says that these two notions of ‚Äúsameness‚Äù for types are the same. ‚Äî‚Äî nLab

    $$ UA^{\mathcal{U}}:\Pi_{(A,B:\mathcal{U})}is\text{-}eqiv(id\text{-}to\text{-}‚âÉ(A,B)) $$

    Identification in Œ£-types

    We can characterize the identity type of Œ£-type as a Œ£-type of identity types.

    Fix an $A$ type, $x: A ‚ä¢ B(x)\ \ type$, $S := Œ£_{(x:A)}B(x)$, we want a reflexive relation $z, z': S ‚ä¢ R(z,z')\ \ type$, we can call this type $Eq\text{-}Œ£$

    Assume

    $$ z ‚âê (x, y) $$

    $$ z' ‚âê (x', y') $$

    By $ap_{pr_{1}}$2 we should have a $p: x \underset{A}= x'$

    Recall the $tr_B$ function we mentioned before, with this, we can define "type of identifications of $y$ with $y'$ over $p$":

    $$ (y \overset{B}{\underset{p}=} y') := (tr_B(p,y) \underset{B(x')}{=} y') $$

    And we can then define $Eq\text{-}Œ£$:

    $$ Eq\text{-}Œ£(z, z') := Œ£_{(p:x=x')} (y \overset{B}{\underset{p}=} y') $$

    Eq-Œ£ : {A : Set} {B : A ‚Üí Set} (z : Œ£ A B) (z' : Œ£ A B) ‚Üí Set
Eq-Œ£ {_} {B} z z' = Œ£ (fst z ‚â° fst z') (Œª p ‚Üí transport B p (snd z) ‚â° snd z')

    To show this is reflexive, we should have a term:

    $$ refl\text{-}Eq\text{-}Œ£ : \Pi_{(z:S)}Eq\text{-}\Sigma(z, z) $$

    This is easy to construct such a term with a pair of refl (notice fst refl and snd refl in this pair is in different types 3):

    refl-Eq-Œ£ : ‚àÄ{A : Set} {B : A ‚Üí Set} (z : Œ£ A B) ‚Üí Eq-Œ£ z z
refl-Eq-Œ£ z = refl , refl

    With path induction4, we can push this a step further to:

    $$ pair\text{-}Eq : \Pi_{(z,z':S)} ((z\underset{S}=z') \rightarrow Eq\text{-}\Sigma(z, z')) $$

    pair-Eq : ‚àÄ{A : Set} {B : A ‚Üí Set} {z : Œ£ A B} {z' : Œ£ A B} ‚Üí
    (z ‚â° z') ‚Üí Eq-Œ£ z z'
pair-Eq {_} {_} {z} refl = refl-Eq-Œ£ z

    And reversely:

    $$ Eq\text{-}pair : \Pi_{(z,z':S)} (Eq\text{-}\Sigma(z, z') \rightarrow (z\underset{S}=z')) $$ 5

    Eq-pair : ‚àÄ{A : Set} {B : A ‚Üí Set} {z : Œ£ A B} {z' : Œ£ A B} ‚Üí
    Eq-Œ£ z z' ‚Üí (z ‚â° z')
Eq-pair (refl , refl) = refl

    And we can find that $Eq\text{-}pair$ and $pair\text{-}Eq$ are "inverse" of each other:

    $$ \Pi_{(z,z':S)} \Pi_{(w:Eq\text{-}Œ£(z,z'))} Eq\text{-}pair(pair\text{-}Eq(w)) = w $$

    $$ \Pi_{(z,z':S)} \Pi_{(r:z=z')} pair\text{-}Eq(Eq\text{-}pair(w)) = w $$

    inv-l : ‚àÄ{A : Set} {B : A ‚Üí Set} {z : Œ£ A B} {z' : Œ£ A B} (w : Eq-Œ£ z z') ‚Üí
    pair-Eq ( Eq-pair w ) ‚â° w
inv-l (refl , refl) = refl

inv-r : ‚àÄ{A : Set} {B : A ‚Üí Set} {z : Œ£ A B} {z' : Œ£ A B} (r : z ‚â° z') ‚Üí
     Eq-pair ( pair-Eq r ) ‚â° r
inv-r refl = refl

    So we can find out that $Eq\text{-}pair$ is a section of $pair\text{-}Eq$ and $pair\text{-}Eq$ is a retraction of $Eq\text{-}pair$.

    Thus we can form an equivalence between $z = z'$ and $Eq\text{-}Œ£(z, z')$:

    $$ (z \underset{S}= z') ‚âÉ Eq\text{-}Œ£(z, z') $$
